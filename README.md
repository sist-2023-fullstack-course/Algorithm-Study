## Commit Rule

- 폴더 이름은 문제사이트*번호*문제이름(같은 문제 폴더에 각자의 파일을 add)
  - `BOJ_1330_두 수 비교하기`
- 파일 이름은 이름.java
  - `윤연상.java`
- Commit message는 [문제사이트]문제번호\_이름
  - `[BOJ]1330_윤연상`
- Commit은 문제 단위로 하고, Commit 한개당 문제 파일 추가 + Readme Problem Table까지 수정
